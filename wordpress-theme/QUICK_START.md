## ⚡ FAST TRACK (3 Steps)

### Step 1: Create ZIP File (1 minute)
```bash
# On Mac/Linux:
cd evolve-merchant-services/wordpress-theme
zip -r evolve-merchant-services.zip evolve-merchant-services/

# On Windows:
# Right-click the "evolve-merchant-services" folder
# → Send to → Compressed (zipped) folder
```

### Step 2: Upload to WordPress (2 minutes)
1. Login to WordPress admin (`yoursite.com/wp-admin`)
2. Go to **Appearance → Themes**
3. Click **Add New** → **Upload Theme**
4. Choose your ZIP file → **Install Now**
5. Click **Activate**

### Step 3: Create Pages (5 minutes)

Create these 6 pages:

| Page Title | Slug | Template |
|------------|------|----------|
| Home | `home` | Default |
| Solutions | `solutions` | Solutions Page |
| Products | `products` | Products Page |
| Referral Partners | `referral-partners` | Referral Partners |
| Privacy Policy | `privacy-policy` | Privacy Policy |
| Terms & Conditions | `terms-and-conditions` | Terms & Conditions |

**How to create a page:**
1. Pages → Add New
2. Enter title
3. Select template from "Page Attributes" box
4. Click **Publish**

### Step 4: Set Homepage (1 minute)
1. Settings → Reading
2. Select **"A static page"**
3. Homepage: **Home**
4. Click **Save Changes**

## 🐛 Troubleshooting

### Theme doesn't appear
**Fix:** Check folder structure. File should be at:
`/wp-content/themes/evolve-merchant-services/style.css`

### Homepage shows blog posts
**Fix:** Settings → Reading → Select static page

### Particle background not working
**Fix:** Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)

### Forms don't work
**Fix:** Forms open your email client by default. This is normal!
To use database forms, install **Contact Form 7** plugin
